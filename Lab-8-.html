4 . Find summary statistics (i.e.need to remember that variances are the same for ANOVA )

# count is combo of group by & n - this is why we just use n here: 

penguin_f_summary <- penguin_f %>% 
  group_by(species) %>% 
  summarize(
    mean_flipper_length = mean(flipper_length_mm),
    sd_flipper_length = sd(flipper_length_mm),
    sample_size = n(),
    se_flipper_length = sd(flipper_length_mm) / sqrt(n()),
    var_flipper_length = var(flipper_length_mm)
  )

Levenes Test (hypothesis test for equal variance):

6. One-way ANOVA

Is there a significant difference in mean flipper length for female penguins between the three species (Adelie, Gentoo, Chinstrap)?

penguin_aov <- aov(flipper_length_mm ~ species, data = penguin_f)
summary(penguin_aov)
##              Df Sum Sq Mean Sq F value Pr(>F)    
## species       2  21416   10708   411.8 <2e-16 ***
## Residuals   162   4213      26                   
## ---
## Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1

Large f value - large between group variance, low within group variance. P values indicate that we dont think these groups came from the same mean. Lots of distance between groups compared to how small they are.

Null hypotheis - means are equal across all groups Alternative hypothesis - means are not equal across all groups (at least two differ)

The mean flipper lengths for the three species (for female penguins) are not equal.

..then which ones differ significantly? Post-hoc pairwise comparisons. Tukeys HSD can be used.

TukeyHSD(penguin_aov)
